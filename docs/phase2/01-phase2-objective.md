# Phase2 開始時の目的（確定原文）

Status: 確定原文

以下は、Phase2 の設計レビュー開始時に TeTe が明文化した目的の原文である。

> Phase 2 の目的は、ランキング品質を改善しながら Phase 1 Stable より悪化しないことです。

## 事実確認による補足

この目的について、後の事実確認で次が確定している。

- 「ランキング品質を改善」という語は、上記の開始時メッセージに明文化されていた。
- 一方で「順位を動かすこと」「Top1やTop3を変えること」「順位変更による改善」は、開始時仕様には明文化されていなかった。
- したがって Phase2 の当初目的は「ランキング品質の改善（対象は当初未定義）＋ Phase1 Stable より悪化しないこと」であり、「順位を動かすこと」を目的として先取りで確定してはならない。

この目的が、後続の Ranking Quality Specification（品質の操作的定義）を必要とした理由である。
